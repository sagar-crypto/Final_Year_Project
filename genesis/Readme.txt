To execute the application:
	1.)run kerberos with node 

	2.)Install node and after installation, type npm init -y to install the library used

	3.)The website will be launched on localhost:3001

	4.)Now run kerberos-auth with node

	5.)save saarthi-main in xampp htdocs and start the apache server and the database

	6.)Before running any application above please import both the databases kerberos.sql and project.sql

	7.)Try accessing localhost:3001, this will open the first page of our application
